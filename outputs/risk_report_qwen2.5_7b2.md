## 2. Identified Assets

- **LLM Model**: The core component of the chatbot responsible for generating responses.
- **System Prompt**: A predefined set of instructions guiding the LLM's behavior.
- **Chat Interface**: The frontend where users interact with the chatbot.
- **Conversation History**: A record of past interactions to provide context for current conversations.
- **Knowledge Base**: Internal documentation used by the chatbot to answer questions.
- **Application Logs**: Records of system activities, including user inputs and generated responses.

## 3. Attack Surface Overview

The main attack surface is the chat interface where users can send arbitrary messages. The attacker has the capability to interact with the chatbot through this interface while being limited by:

- No direct access to server components.
- No direct access to model weights or database.

Potential points of attack include:

1. **Prompt Injection**: Manipulating the system prompt to alter the behavior of the LLM.
2. **Privacy Leakage**: Extracting sensitive information from the chatbot's responses or conversation history.
3. **Hallucination/Misinformation**: Inducing the chatbot to provide incorrect or misleading answers.
